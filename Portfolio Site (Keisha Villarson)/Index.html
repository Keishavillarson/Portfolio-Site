The Expert App

UX Case study of an expert pairing app

Overview

The goal of this app is to give people a simple, intuitive way to connect with an expert in nearly any field within seconds so they can feel more informed and more prepared to face their everyday (and not-so-everyday) problems.

Problem

Our users need a way to access experts in order get their questions answered in a time of need. We will know this to be true when we see users effectively completing their personal projects with expert advice.

Research Goals

To better understand user behavior and learning To determine which tasks users would like to complete while using a learning app / site. Documenting user pain points with existing learning apps / sites.
